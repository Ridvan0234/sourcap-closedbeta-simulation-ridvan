# Environment Setup (RAP)

You need a working ABAP environment that supports RAP.

## Minimal checklist
- Git installed
- GitHub account (mandatory)
- ADT (Eclipse + ABAP Development Tools)

## ABAP environment options
**Option A (recommended):** SAP BTP Trial (ABAP Environment)  
**Option B:** S/4HANA system with RAP support (1909+)

## ADT connection (high-level)
1) Open Eclipse/ADT
2) Create an ABAP Cloud Project
3) Log in to your system/trial
4) Create a package: `Z_BETA_STOCK_<YOURNAME>`
5) Keep objects clean and named
