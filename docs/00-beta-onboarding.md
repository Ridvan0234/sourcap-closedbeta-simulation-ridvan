# Closed Beta Onboarding

## Purpose
This simulation validates **delivery readiness**: can you turn a small business ask into a clean RAP deliverable (with AI help) and prove it?

## Expected time
**2–4 hours**. If you’re stuck longer: open a GitHub Issue (Question template).

## Mentor review mindset
We value:
- Clarity over complexity
- Process over perfection
- Honest explanation (AI is allowed — understanding is required)
